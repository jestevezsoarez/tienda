Tecnologías utlizadas

Este es un proyecto realizado con fines de estudio

Autor: Javier Estévez

Angular versión 13

Se usaron componentes a fin de dividir la pagina en piezas de código reutilizables.

Este sitio web es una SPA (Single Page Application) y por lo tanto hace uso de rutas ejecutadas desde el menú de navegación. Se utilizan interfaces, servicios para reemplazar información en la página web. También se hace uso de animaciones css utilizando parte de la librería animate.css

Firebase

Utilizo Firebase para crear la base de datos y luego cargarla desde la página.

HTML5 | CSS3

Tecnologías fundamentales para dar estructura a la página y posterior aspecto visual.

Node.js

En este sitio utilizo Node.js para descargar paquetes de mapas de ciudades mediante el NPM (Node Package Manager)

Dichos mapas son utlizados en las sucursales obteniendo coordenadas de longitud y latitud desde la base de datos.

Typescript

Uso de Typescript como lenguaje de programación de Angular que luego es transpilado a Javascript.
